Convert Greek to Latin (ELOT743) για QGIS - Οδηγίες εγκατάστασης σε Windows
---------------------------------------------------------------------------

Το script "Convert Greek to Latin (ELOT743)" μετατρέπει ελληνικές αναγραφές στις αντίστοιχες λατινικές,
βάσει των προδιαγραφών του προτύπου ΕΛΟΤ 743.


Οδηγίες εγκατάστασης:
---------------------

1. Πλοηγηθείτε στον φάκελλο: C:\Users\User1\.qgis2\processing\scripts, όπου "User1" το όνομα Χρήστη των Windows.
   Εναλλακτικά, μπορείτε να εντοπίσετε αυτόν τον φάκελλο ως εξής:
   α. Προσθέστε ένα έτοιμο script του QGIS: Processing Toolbox (Advanced Interface) \ Scripts \ Tools \ Get scripts from on-line scripts collection.
   β. Κάντε δεξί κλικ στο script \ Edit script \ Save as... \ και αντιγράψτε τη διαδρομή δίσκου όπου βρίσκεται το script.

2. Αντιγράψτε σε αυτόν τον φάκελλο τα αρχεία:
   - Convert_Greek_to_Latin_ELOT743.py
   - Convert_Greek_to_Latin_ELOT743.py.help
   - icpyqg.py

3. Ανοίξτε με έναν διορθωτή κειμένου κατάλληλο για script editing (π.χ. Notepad++, Pythonwin) το αρχείο
   "Convert_Greek_to_Latin_ELOT743.py" και στην γραμμή 28 θέστε την διαδρομή που εντοπίσατε:

   theModulePath = "C:\\Users\\User1\\.qgis2\\processing\\scripts"

   Η διαδρομή αυτή χρειάζεται για τον εντοπισμό του module "icpyqg" (οι διπλές κάθετοι "\\" απαιτούνται από την Python).

4. Ανοίξτε το QGIS. To script "Convert Greek to Latin (ELOT743)" θα πρέπει να φαίνεται στην υπο-ομάδα "Table"
   των Scripts του Processing Toolbox (Advanced Interface).

5. Τρέξτε το script με διπλό κλικ. Επιλέξτε τον πίνακα και τα πεδία ελληνικών - λατινικών.
   Στο tab "Help" υπάρχουν σύντομες οδηγίες χρήσης.


Παρατηρήσεις:
-------------
α. Το script αυτό επιδρά στις επιλεγμένες εγγραφές του πίνακα. Αν δεν υπάρχουν επιλεγμένες εγγραφές, τότε επιδρά σε όλες.
β. Ο Script Editor του Processing Toolbox (δεξί κλικ \ Edit script) δεν αποθηκεύει σωστά τους ελληνικούς χαρακτήρες.
   Εάν το script περιέχει ελληνικούς χαρακτήρες (π.χ. το "icpyqg.py"), τότε μην το επεξεργάζεστε με αυτόν τον editor,
   αλλά με άλλον κατάλληλο για script editing (π.χ. Notepad++, Pythonwin) ή με τον editor της Python Console του QGIS.
